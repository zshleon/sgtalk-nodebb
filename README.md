# SGTALK NodeBB

SGTALK is a NodeBB-based Singapore Chinese newcomer forum customization project.

This repository is the canonical code handoff source for theme, deployment notes, and QA documentation. It does not contain production secrets, MongoDB data, NodeBB runtime data, uploads, or backups.

Start here:

- `HANDOFF.md` for current stack, deploy commands, and takeover prompt
- `docs/SGTALK_PHASE1_QA.md` for verified UI and interaction status
- `docs/SGTALK_V2EX_UI_BLUEPRINT.md` for the V2EX-style product target
- `docs/sgtalk-v2ex-design-board.html` for the visual design board
- `themes/nodebb-theme-sgtalk-v2ex` for the custom theme

Live site: https://sgtalk.zshstc.org
