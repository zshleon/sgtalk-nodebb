发布状态
1. 可编辑

标题和正文为空时，发布按钮可点击但会触发 inline validation；或直接禁用并显示原因。

2. 提交中

按钮显示提交中并禁用，textarea 不被永久锁定。

3. 成功

跳转到新主题详情页。

4. 失败

显示错误，按钮恢复，内容保留。

This is the acceptance contract for every publish-related bug fix.
